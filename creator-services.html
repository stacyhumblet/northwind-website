Custom Work

How We Scope Changes & Add-Ons

Every change request beyond the core product is classified by size. Smaller changes are priced hourly. Larger changes are scoped as projects.

Small — Hourly Rate

Quick Changes & Adjustments

Targeted modifications that don't require structural changes to your dashboard or data model.

  • Add or rename a KPI or metric
  • Add a new chart or visualization to an existing dashboard
  • Rename or reorder data fields
  • Small data mapping adjustments
  • Change a filter or date range default
  • Update color scheme or label styling
Medium — Scoped Project

New Dashboards & Data Logic

Changes that require new build work — additional dashboards, data source additions, or custom calculation layers.

  • Build a new dashboard (e.g. Campaign ROI, Cohort Analysis)
  • Add a new data source to an existing integration
  • Custom attribution model (first-click, multi-touch, time-decay)
  • Custom segment or audience filter logic
  • Automated email or Slack report delivery
  • New cross-platform join or combination view
Large — Full Project Scope

New Platforms & Advanced Systems

Full-scope projects involving new platform integrations, predictive analytics, or custom data infrastructure.

  • Integrate a new platform (e.g. YouTube, Pinterest, Amazon)
  • Custom data warehouse setup
  • Predictive modeling (forecasting, churn, LTV prediction)
  • Custom API development or data pipeline engineering
  • Full white-label branded dashboard build
  • Enterprise-level multi-user access & permissions
